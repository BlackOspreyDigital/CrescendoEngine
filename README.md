# CrescendoEngine BRAVO

Built with Vulkan and SDL2

BRAVO Build serves as the latest rolling dev build for Crescendo. This is where new systems and refactors will take place before being rolled out to the main Branch. The last round of updates included adding compositer and post-processing pipelines allowing for shaders to really shine with emissive blooms and blur.
